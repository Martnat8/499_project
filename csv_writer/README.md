# CSV Writer

A **CSV Writer** node that subscribes to topics and saves the data to a csv.

## Parameters

| Name              | Type    | Default             | Description                                        |
|-------------------|---------|---------------------|----------------------------------------------------|
| `topic_in`        | string  | `/raw_image_out`    | A topic to run the filter on                       |
